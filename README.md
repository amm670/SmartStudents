# SmartStudents
We believe in using technology to connect all college students across the country so that, together, we can achieve our full potential. The way we do that is by creating a mobile marketplace that is beautifully designed, fun to use and enables students to save money by exchanging all kinds of stuff on campus.


Download the starter project files as .zip and extract to your desktop.
